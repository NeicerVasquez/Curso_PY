# Game Proyect

Para correr el juego debe seguir las siguientes instrucciones en la terminal:
```sh
cd game
python3 main.py
```

# ReadCSV Proyect
Para Linux
```sh
git clone
cd readCSV
mkdir imgs
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```

Para Windows
```sh
git clone
cd readCSV
mkdir imgs
python3 -m venv env
source env/Scripts/activate
pip3 install -r requirements.txt
python main.py
```
